Book Description

Create, merge, and distribute code with Git, the most powerful and flexible versioning system available

In Detail

Since its inception, Git has attracted skilled developers due to its robust, powerful, and reliable features.
 Its incredibly fast branching ability transformed a piece of code from a niche tool for former Linux Kernel 
 developers into a mainstream distributed versioning system. Like most powerful tools, Git can be hard to
 approach since it has a lot of commands, subcommands, and options that easily confuse newcomers. The main 
 purpose of Git Essentials is to help you overcome this fear and become adept at all basic tasks in Git. 
 This book starts with an introduction to version control systems, and after this you'll delve deeply into 
 the essentials of Git. This serves as a primer for the topics to follow such as branching and merging, 
 
 creating and managing a GitHub personal repository, and fork and pull requests. You will then learn the 
 art of cherry-picking, taking only commits you like, followed by Git blame. Finally, you'll learn how to 
 migrate from SVN using Git tools or TortoiseGit and migrate from other VCSs, concluding with a collection
 of resources, links, and appendices.

　　原标题：水位比当年溃口时仅低2米多 18年后千人上堤死守簰洲湾

　　楚天都市报记者 周鹏 通讯员 江开群 谌胜蓝

　　簰洲湾，一个承载了许多悲伤与感动记忆的名字。1998年8月1日晚，在那场世纪大洪水中，咸宁市嘉鱼县簰洲湾江堤出现溃口，小镇一片汪洋，高建成等19名烈士献身于此。

　　18年后，洪水再次肆虐。堤外，长江水位直逼1998年，簰洲湾段已超警戒水位；　堤内，335毫米的强降雨，令全镇6.3万多亩农田全部被淹。

　　外洪内涝，形势紧迫。昨日，楚天都市报记者赶到簰洲湾镇走访。

　　水位超警戒，6万亩农田被淹

　　昨日上午，记者驱车从嘉鱼县潘湾镇驶往簰洲湾镇。翻过沙湖堤，看到路边干渠里的洪水漫出，两边农田都泡在水中，成堆的南瓜浮在水面。

　　东岭村村民李仕平今年种了20亩水稻，如今全部被淹。他说，如果洪水没顶，几天排不掉，水稻将绝收。

　　“可以说是外洪内涝！”簰洲湾镇领导干部大都上堤去了，留守的镇长宗克飞介绍，到昨日早上8时，簰洲湾本轮强降水达335毫米，湾内一片汪洋，全镇6.3万多亩农田全部不同程度被淹。幸好，昨日没有继续降雨。

　　而在堤外，长江汛情同样严峻。昨日上午8时，簰洲湾长江水位已达28.52米，超警戒水位0.02米，水位还在不断上涨。

　　不能让洪水惊扰烈士坟墓

　　记者来到簰洲湾大堤上，看到这里已搭起帐篷。堤内有人在巡视，有人在割草。

　　在中堡村段，46岁的村治保主任童方月正在带队巡查。他回忆说，1998年发洪水时，村民和解放军战士一起上堤抢险，他也帮忙扛沙袋、堵管涌，没日没夜地干，不过没能阻止溃堤。

　　“现在的堤牢固多了！”童方月带记者来到当年的溃堤处。1998年洪灾后，当地政府重新修固江堤，现在的大堤比原来高了2米、宽了3米，并且进行了硬化。童方月介绍，当年溃口时的水位是31米多，而昨日的水位不到29米，目前江堤还比较安全。

　　堤内不远处有一个墓园，那里长眠着当年为抢险献身的19位解放军战士，每年村民们都会去祭奠。“我们决不能让洪水淹了他们的墓。”童方月说。

　　1200多村民日夜上堤巡防

　　据介绍，长江簰洲湾堤共有41.5公里长。2日，咸宁市、嘉鱼县和簰洲湾镇三级党委政府领导到堤上召开现场会，要求严防死守，以防万一。

　　目前，咸宁市、嘉鱼县已启动防汛四级预案，嘉鱼县领导干部全部上堤巡防。宗克飞介绍，该镇按照每公里30人的标准巡防，参与巡防的村民共有1200多人。

　　335毫米的降水要多久能排完？宗克飞说，该镇共修建了三个大的泵站，39台机组全部开启，大约需要9天，才能将这些雨水排入长江。

　　为保障防汛供电，咸宁市电力公司紧急调来一台315千伏变压器，沿堤架起临时电路，为97个哨所照明。

　　目前，簰洲湾镇排涝防洪工作正有序进行。
